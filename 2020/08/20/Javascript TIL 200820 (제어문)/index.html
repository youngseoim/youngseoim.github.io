블록문

  1. 0개 이상의 문을 중괄호로 묶은 것, 코드 블록 또는 블록.
  2. 자바스크립트는 블록문을 하나의 실행 단위로 취급한다, 제어문이나 함수를 정의할 때 사용하는 것이 일반적이다.
  3. {var foo = 10;} == 문의 끝에는 세미콜론을 붙이는 것이 일반적이지만 블록문은 자체 종결성을 갖기 때문에 붙이지 않는다.

조건문

  1. 조건문은 주어진 조건식의 평과 결과에 따라 코드 블럭의 실행을 결정한다
  2. 조건식은 불리언 값으로 평가될 수 있는 표현식이다. if..else문 , switch문

자바스크립트의 두 가지 조건문

  1. if..else문

    1. if문의 조건식은 불리언 값으로 평가 되어야 한다 .

    2. if문의 조건식이 불리언 값이 아닌 값으로 평가되면 자바스크립트 엔진에 의해 암묵적으로 불리언 값으로 강제 변한되어 실행할 코드 블록을 결정한다 .

    3. 조건식을 추가하여 조건에 따라 실행될 코드 블록을 늘리고 싶으면 else if문을 사용한다

    4. Else if 문과 else문은 옵션이다,if와 else문은 2번 이상 사용할 수 없지만 else if 문은 여러 번 사용할 수 있다.

    5. 대부분의 if..else문은 삼항 조건 연산자로 바꿔 쓸 수 있다.

  2. Switch

    1. 주어진 표현식을 평가하여 그 값과 일치하는 표현식을 갖는 case 문으로 실행 흐름을 옮긴다.

    2. default 문은 일치하는 case문이 없다면 실행되는 문이고 선택사항이다

    3. break가 존재하지 않으면 실행이 되야하는 문부터 계속 내려가기 때문에 꼭 필요하다 , 이를 폴스루라고 한다

반복문

  1. 반복문은 조건식의 평가 결과가 참인 경우 코드 블록을 실행하고 조건식이 거짓일 때 까지 반복된다

  2. 세 가지 반복문인 for, while, do..while문을 제공한다.’

  3. **반복 횟수가 명확할땐 for, 명확하지 않을 때 while, **

  4. for문

    1. 조건식이 거짓으로 평가 될 때 까지 코드 블록을 반복 실행한다
    2. 실행 순서

  5. while문

    1. 주어진 평가 결과가 참이면 반복해서 실행한다

    2. 무한 루프에서 탈출 하기 위해서는 if 문으로 탈출 조건을 만들고 break문으로 코드 블록을 탈출한다

  6. do..while문

    1. 코드 블록을 먼저 실행하고 조건식을 평가한다, 따라서 무조건 한 번 이상 실행 된다 .

  7. break문

    1. 레블 문, 반복문, switch 문의 코드 블럭을 탈출한다 이외에 문에 사용하면 문법 에러가 발생한다 .

  8. continue문

    1. 현 지점에서 코드 블록 실행을 중단하고 반복문의 증감식으로 실행 흐름을 이동 시킨다 .
