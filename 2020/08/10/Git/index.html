Git 강의

  1. $ 입력받을 준비가 됐다 없다면 뭔가가 실행중인것
  2. ~ 최상위 폴더
  3. Ls -a/l a는 숨김
  4. Clear
  5. pwd 현재 쓰고 있는 디렉토리를 절대경로로 출력
  6. Echo “” 따옴표 안에 내용 출력
  7. Mkdir 디렉토리명 디렉토리(폴더) 생성
  8. Touch 파일명 엑셀같은 애들은 생성 불가
  9. Mv(move) style.css(파일명) bin(위치)
  10. cp(copy) style.css(파일명) ..(상위폴더) 복사
  11. copy 사본의 이름 , move 위치이동 하면서 이름변경(rename) mv style.css index.js
  12. rm(remove삭제) rm index.* > index 들어간거 다 삭제
    1. rm *.py py 확장자 다삭제
    2. rm은 폴더삭제는 불가능
    3. rm -r 안에 오브젝트를 다 지우고 자기자신까지 삭제 (오브젝트가 남아있으면 폴더가 지워지지 않는다 경로를 잃기 떄문에)
    4. / 경로에서는 어떤 것도 삭제하면 안된다
    5. sudo키워드를 쓸 땐 꼭 잘 확인할 것
  13. chmod 777 style.css 권한부여 0부터 8까지 8진수를 3개의 숫자로 권한
  14. Drwx(나)rwx(유저그룹)rwx(게스트) rwx 는 421 원하는걸 더한다 r,w만 키고 싶다면 4,2를 더해서 chmod 666 파일명
  15. open 파일명 기본확장자로 열어준다
  16. Cat READE.me = 안의 내용을 터미널에 텍스트로 제공받는다.
  17. Gitignore 깃에 올라갈 때 제외시키는 목록

Vim

  1. i = insert
  2. 샵 = h1 , 샵샵 = h2
  3. 데쉬 = ul 태그 li 태그
  4. 1.2.3. = ol태그
  5. p = 그냥 텍스트 작성
  6. ` 3개에 양식 작성 후 코드작성
  7. 대괄호go to google대괄호(주소) =링크
  8. 느낌표 대괄호macbook with iPad] (이미지경로) = 이미지 삽입
  9. y (복사) p (붙여넣기)
  10. u 누르면 삭제
  11. dd 잘라내기
  12. 붙여넣기 p
  13. 컨트롤 r 되돌리기
  14. 쉬프트 + 세미콜론 = 상단 메뉴바 소환
  15. w = 저장
  16. q = 나가기, q! = 처음상태를 덮어쓰고 나간다 wq = 저장후 종료

Git

  • Git 초기 설정
    • Git username, email
    • git config –global core.editer “vim”
    • git config –global core.pager “cat”
