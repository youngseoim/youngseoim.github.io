JSX

  1. JSX란?

    1. Javascript를 확장한 문법
    2. React에서는 이벤트가 처리되는 방식, 시간에 따라 상태가 변하는 방식, 렌더링 되기 전 데이터가 준비되는 방식등 렌더링 로직이 본질적으로 다른 UI로직과 연결된다.
    3. React는 기존의 별도의 파일에 html,css,Js를 인위적으로 분리하는 대신 둘 다 포함하는 “컴포넌트”의 개념을 사용해서 한개의 파일에서 html,css,js를 작성해서 하나의 유닛 처럼 사용한다.
    4. JSX는 Element를 생성한다.
